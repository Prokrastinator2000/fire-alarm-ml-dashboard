# 🔥 Fire Alarm ML Dashboard

Веб-приложение для предсказания срабатывания пожарной сигнализации 
на основе показаний датчиков.

## Модели
- DecisionTreeClassifier
- GradientBoostingClassifier  
- LightGBMClassifier
- BaggingClassifier
- StackingClassifier
- XGBoostClassifier

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
